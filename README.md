### Assignment details  
please fix this very ugly code!  
Create a new file in your Text Editor (Atom, Sublime) and properly indent the code below:

```
<!DOCTYPE html>
<html>
<head>
<title>
Basic I
</title>
</head>
<body>
<h1>
What language do you love?
</h1>
<p>
I love HTML!
</p>
</body>
</html>
```
